# Virtual-Museum

nasa pre-colonial at pre-colonial lit ang templates.
if video ang ilalagay instead na img dun sa cards, palitan nyo lang like this:
```html
<video src="/assets/images/blah.mp4" muted autoplay controls></video>
```
kung audio naman:
```html
<div class="audio-container">
    <audio src="/assets/images/blah.mp3" controls>
</div>
```
